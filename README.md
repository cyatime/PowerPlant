# 杂货屋

## 介绍

用来构建电视和影片资源的网站。

## 初始化

```bash
$ yarn
```

## 如何运行

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## 测试

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
